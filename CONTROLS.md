# 游戏控制说明 (Game Controls)

## 主菜单按键说明

### 1. 开始旋转轮盘
- **按键**: `SPACE` (空格键)
- **状态**: MENU
- **作用**: 开始旋转轮盘，随机选择一个未玩过的游戏

### 2. 启动选中的游戏
- **按键**: `ENTER` / `RETURN` (回车键) ⚠️ **不是空格键！**
- **状态**: WAITING (轮盘停止后)
- **作用**: 启动轮盘选中的游戏

### 3. 退出游戏
- **按键**: `ESC` (退出键)
- **状态**: FINAL (最终分数界面)
- **作用**: 关闭游戏

## 常见问题

### Q: 为什么按空格键无法启动游戏？
**A**: 在WAITING状态下，需要按 **ENTER键（回车键）** 而不是空格键！
- 空格键 (SPACE, 键码32) - 用于开始旋转轮盘
- 回车键 (ENTER/RETURN, 键码13) - 用于启动游戏

### Q: 如何区分这两个键？
**A**: 
- **空格键**: 键盘最下方最长的那个键
- **回车键**: 键盘右侧，通常标有 "return" 或 "enter"，或者有一个向左的箭头符号 ↵

## 调试信息

如果按键没有响应，请查看终端输出：
- 正常情况下会显示: `检测到按键: key=13, 当前状态=WAITING`
- 如果显示 `key=32`，说明您按的是空格键，而不是回车键

## 游戏流程

1. **MENU** 状态 → 按 `SPACE` → 开始旋转
2. **SPINNING** 状态 → (自动) → 轮盘旋转
3. **WAITING** 状态 → 按 `ENTER` → 启动游戏 ⚠️
4. **PLAYING** 状态 → (游戏进行中) → 输入获胜者
5. **MENU** 状态 → 重复步骤1-4
6. **FINAL** 状态 → 按 `ESC` → 退出游戏
