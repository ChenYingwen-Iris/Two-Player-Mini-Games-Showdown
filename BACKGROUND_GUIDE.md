# 添加背景图片说明

## 📸 如何添加背景图片

### 步骤1: 保存背景图片

请将你的背景图片（森林像素风图片）保存到项目根目录，文件名可以是以下任一：
- `background.png` (推荐)
- `background.jpg`
- `background.jpeg`
- `bg.png`

### 步骤2: 确认文件位置

确保图片在正确的位置：
```
Two-Player-Mini-Games-Showdown/
├── background.png          ← 你的背景图片
├── game_launcher.py
├── yellowbox.png
├── redbox.png
├── bluebox.png
├── pinkbox.png
├── greybox.png
└── ...
```

### 步骤3: 运行游戏

```bash
python game_launcher.py
```

## ✨ 功能说明

程序会自动：
1. 尝试加载背景图片
2. 自动缩放到 800x480 分辨率
3. 如果找不到图片，使用默认深灰色背景

## 🎨 背景效果

背景图片会应用到：
- ✅ 主菜单界面（轮盘界面）
- ✅ 旋转动画界面
- ✅ 游戏选择界面
- ✅ 最终分数界面

## 🖼️ 推荐的图片格式

- **格式**: PNG 或 JPG
- **建议尺寸**: 800x480 或更大（会自动缩放）
- **风格**: 像素风格森林背景（如你提供的图片）

## 📝 终端输出

成功加载背景时，你会看到：
```
✓ 成功加载背景图片: background.png
✓ 成功加载箱子图片!
```

未找到背景图片时：
```
⚠ 未找到背景图片，使用默认背景
✓ 成功加载箱子图片!
```

## 💡 提示

- 图片会自动拉伸填充整个窗口（800x480）
- 建议使用横向的像素风格图片
- 如果背景太亮，文字可能不清晰，可以考虑：
  - 使用稍暗的背景图
  - 或在代码中添加半透明遮罩层

---

**现在请将你的森林背景图片保存为 `background.png` 放在项目根目录，然后运行游戏即可！** 🎮🌲
