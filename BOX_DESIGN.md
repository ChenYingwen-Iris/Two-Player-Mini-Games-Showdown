# 箱子轮盘设计说明

## 📦 新设计概览

游戏轮盘已改为 **四个箱子 (Box)** 的设计，更加直观和美观！

## 🎨 箱子布局

```

│                                     │
│         GAME SHOWDOWN               │
│                                     │
│     ┌─────────┐   ┌─────────┐      │
│     │ 黄色箱子 │   │ 红色箱子 │      │
│     │数蝴蝶   │   │双人迷宫  │      │
│     └─────────┘   └─────────┘      │

```

## 🎯 箱子分布

| 位置 | 颜色 | 游戏 | 未玩颜色 | 已玩颜色 |
|------|------|------|----------|----------|
| 左上 | 黄色 | 数蝴蝶 | (255, 200, 0) | 灰色 |
| 右上 | 红色 | 双人迷宫 | (255, 80, 80) | 灰色 |
| 左下 | 蓝色 | 硬币收集 | (80, 120, 255) | 灰色 |
| 右下 | 粉色 | 拔河大战 | (255, 100, 200) | 灰色 |

## 🔄 指针旋转机制

### 指针角度对应关系

- **左上 (黄色)**: 225° (西北方向)
- **右上 (红色)**: 315° (东北方向)
- **左下 (蓝色)**: 135° (西南方向)
- **右下 (粉色)**: 45° (东南方向)

### 旋转效果

1. 按 SPACE 开始旋转
2. 指针快速旋转 3-5 圈
3. 逐渐减速
4. 精确停在某个箱子的角度上
5. 该箱子对应的游戏被选中

## 📦 箱子设计细节

每个箱子包含：

```
┌─────────────────┐  ← 深色顶部（3D效果）
│                 │
│    ─────────    │  ← 装饰线
│                 │
│       ▓▓▓       │  ← 中央锁扣
│                 │
└─────────────────┘
     游戏名称          ← 箱子下方文字
```

### 箱子元素

- **主体**: 彩色矩形，圆角边框
- **顶部**: 深色矩形（颜色-40），营造3D效果
- **锁扣**: 中央黑色小矩形
- **边框**: 4px 黑色边框
- **文字**: 箱子下方显示游戏名称

## 🎨 颜色状态

### 未玩过的箱子
- 显示鲜艳的原色
- 文字为黑色
- 可以被指针选中

### 已玩过的箱子
- 显示灰色 (120, 120, 120)
- 文字为灰色
- 不会被指针选中

## 🎯 指针设计

### 指针样式
- **主体**: 红色线条 (6px 宽)
- **箭头**: 红色三角形
- **中心点**: 黄色圆点
- **长度**: 60 像素

### 指针运动
```
初始速度: 25°/帧
减速公式: speed = max(0.5, distance / 40)
最终停止: 精确对准箱子角度
```

## 🔧 技术实现

### 箱子位置计算
```python
box_size = 100       # 箱子大小
spacing = 30         # 箱子间距
offset = (box_size + spacing) // 2

positions = [
    (center_x - offset, center_y - offset),  # 左上
    (center_x + offset, center_y - offset),  # 右上
    (center_x - offset, center_y + offset),  # 左下
    (center_x + offset, center_y + offset)   # 右下
]
```

### 指针角度计算
```python
angle_rad = math.radians(pointer_angle)
end_x = center_x + length * math.cos(angle_rad)
end_y = center_y - length * math.sin(angle_rad)
```

## 🎮 用户体验

### 优势
✅ 视觉清晰 - 四个箱子一目了然
✅ 状态明确 - 灰色箱子表示已玩过
✅ 动画流畅 - 指针旋转有减速效果
✅ 反馈直观 - 指针精确指向选中的箱子

### 操作流程
1. 查看四个彩色箱子
2. 按 SPACE 旋转指针
3. 观看指针旋转并减速
4. 指针停止在某个箱子上
5. 该箱子对应的游戏被选中
6. 玩完游戏后，箱子变灰
7. 下次旋转时，灰色箱子不会被选中
